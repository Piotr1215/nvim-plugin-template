1. nvim-plugin-template       *nvim-plugin-template.nvim-nvim-plugin-template*

neovim plugin template integration test and doc publish


USAGE                   *nvim-plugin-template.nvim-nvim-plugin-template-usage*

click `use this template` button generate a repo on your github. then clone
your plugin repo.open terminal then cd plugin directory , run `python3
rename.py your-plugin-name` this will replace all `nvim-plugin-template` to
your `pluing-name`. Then develope your plugin. Enjoy!


TEST                     *nvim-plugin-template.nvim-nvim-plugin-template-test*

use vusted for test install by using `luarocks --lua-version=5.1 install
vusted` then run `vusted test` for your test cases


CI                         *nvim-plugin-template.nvim-nvim-plugin-template-ci*
